Projects

Personal Projects

Wizard Home Golang

Wizard Home is a command-line tool developed in Go for managing and synchronizing configuration files across Linux environments. It simplifies the process of setting up a personalized system by allowing users to store, track, and restore configuration files (e.g., nvim, alacritty, zsh) using a single JSON config file and a remote Git repository.

The tool supports saving and restoring previous states, as well as installing configurations from other users via Git. It is designed to help developers quickly configure new systems from scratch.

This project uses advanced programming concepts such as higher-order functions and file manipulation. A new version with improved concurrency is currently in development. You can view my personal configuration example here.

Cart RESTful API Golang

A complete shopping cart REST API built in Go using the Gin web framework and MongoDB for storage. This service is designed to be easily integrated into any backend system that needs shopping cart functionality.

The project is fully documented and follows the Test-Driven Development (TDD) approach, with over 80% test coverage including unit and end-to-end tests.
